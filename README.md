# QuickApply — Universal Job Application Autofill

QuickApply is a Chrome browser extension that eliminates the repetitive pain of filling out the same job application fields over and over. Save your personal details, work history, education, and links once — then click "Fill Form" on any job application page to instantly populate matching fields. Works across major Applicant Tracking Systems (ATS) as well as generic HTML forms.

## Key Features
- **One-time profile setup:** Stored locally in your browser for total privacy.
- **Smart field detection:** Uses labels, names, ids, and placeholders to intelligently map fields.
- **Pre-built ATS matching rules:** Optimized for major platforms including Workday, Greenhouse, and Lever.
- **Application tracker:** Automatically logs which companies and roles you've applied to and when.
- **100% Local Storage:** No data leaves your browser. No backend server, no tracking.

## Installation (Local Developer Mode)
1. Clone or download this repository.
2. Open Chrome and navigate to `chrome://extensions`.
3. Enable **Developer mode** in the top right corner.
4. Click **Load unpacked** and select the `QuickApply` repository folder.
5. Pin the QuickApply extension to your toolbar for easy access!

## Usage
1. Click the QuickApply icon and select **Edit Profile / Tracker** to fill out your job application profile.
2. Navigate to a job application form (e.g., on Greenhouse or Workday).
3. Click the QuickApply extension icon and click **Fill This Page**.
4. The extension will automatically populate all matching fields! Note: For security reasons, file uploads (like Resumes) must always be attached manually.
